</hr>

## API Guide

- [x] POST
- [x] DELETE
- [-] PUT*

## Build
Install the dependencies and start the server application.

```sh
cd backend
npm i
node server.js
```

Verify the deployment by navigating to your web address in
your preferred browser.

```sh
127.0.0.1:8000
```
> Note: 
> Run the `frontend` utilize its functionalities.
